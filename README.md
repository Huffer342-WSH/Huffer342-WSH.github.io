## 使用方法

### git clone
文章和一些代码放在一个子库里 https://github.com/Huffer342-WSH/myNotes ， 所以要加 ```--recursive```

```shell
git clone --recursive https://github.com/Huffer342-WSH/Huffer342-WSH.github.io.git
```

### 安装 Node.js

[https://nodejs.org/en](https://nodejs.org/en)

### 安装packages

```shell
npm i -g pnpm
```

```shell
pnpm i
```

### 更新packages


```shell
pnpm up --latest -i
```
